i. Penjelasan Singkat Algoritma Program

Program penyusunan pengambilan mata kuliah sebagai tugas kecil 2 ini dibuat 
dengan memiliki pendekatan terhadap algoritma toposort yang melakukan 
algoritma decrease and conquer. Algoritma decrease and conquer dapat dipisah menjadi
2 komponen penting, yaitu:
	1. Proses decrease :	proses mengurangi node yang ada hingga tidak tersisa lagi
				sebuah node di dalam graf (dalam hal ini saya menggunakan
				data type dictionary di dalam Python),
	2. Proses conquer  :	proses mencari node-node yang tidak memiliki edge yang
				masuk ke dalam node tersebut di dalam graf berarah.

ii. Requirement Program

Program ini tidak membutuhkan requirement yang rumit. Cukup memiliki setidaknya Python3
maka program ini sudah dapat dijalankan di komputer anda.

iii. Cara Menggunakan Program

Cukup run program layaknya file python lainnya, kemudian masukkan file permasalahan
yang ingin diselesaikan (file test : txtfile1 - txtfile8.txt), lalu setelah
menunggu beberapa saat, hasil penyelesaian berupa daftar mata kuliah yang diambil
akan dikeluarkan oleh program sebagai output.
